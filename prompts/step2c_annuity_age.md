# Step 2c: 연금개시나이 추출

아래 문서에서 각 상품별 **연금개시나이** 정보를 추출하세요.

## 문서
```
{{DOCUMENT}}
```

## 상품 목록 (Step 1 결과)
```json
{{PRODUCT_LIST}}
```

## 추출 규칙

### 1. 연금상품 판별
- 상품명에 "연금"이 포함된 경우에만 연금개시나이를 추출합니다.
- 비연금 상품은 빈 배열을 반환하세요.

### 2. 연금개시나이 찾는 위치
- "연금개시나이" 섹션 또는 해당 키워드가 포함된 테이블
- 보통 "연금개시나이" 테이블에 연금유형별/성별 나이 범위가 있음

### 3. 연금유형별 구분
문서에 아래와 같은 연금유형이 구분되어 있으면 각각 추출:
- 종신연금형, 확정기간연금형, 상속연금형
- 거치형, 적립형
- 개인형, 신부부형
- 기타 문서에 명시된 유형

### 4. 성별 구분
- 남자/여자에 따라 나이 범위가 다르면 각각 별도 행으로
- 성별 구분이 없으면 "성별" 필드는 빈 문자열

### 5. 나이 범위
- "45세~80세" → min=45, max=80
- "만 55세" (단일 값) → min=55, max=55
- 범위를 개별 나이로 확장하지 **마세요**. min/max만 기록하세요.

```json
        "연금유형": "종신연금형"
      }
    ]

  }

```

## 예시

문서에 아래 테이블이 있을 때:
| 구분 | 종신연금형 | 확정기간연금형 |
|------|----------|------------|
| 남자 | 45세~80세 | 45세~80세 |
| 여자 | 45세~75세 | 45세~80세 |

상품명이 "한화생명 바로연금보험 무배당 종신연금형 종신플랜"인 경우:

```json
{
  "상품명": "한화생명 바로연금보험 무배당 종신연금형 종신플랜",
  "보기개시나이정보": [
    {"나이_min": 45, "나이_max": 80, "성별": "남자", "연금유형": "종신연금형"},
    {"나이_min": 45, "나이_max": 75, "성별": "여자", "연금유형": "종신연금형"}
  ]
}
```

주의:
- 연금유형은 상품의 세부종목과 매칭하세요. "종신연금형" 상품에는 종신연금형 데이터만 적용합니다.
- 상품 목록의 **모든 상품**에 대해 결과를 생성하세요 (비연금 상품은 빈 배열).
