## 추출 규칙

### 1. 보험기간 형식
문서에 나오는 보험기간 표현을 **그대로** 기록하세요:
- 연수 기반: "1년", "5년", "10년", "20년" 등 (보통 "N년만기"로 표현)
- 나이 기반: "60세", "70세", "80세", "90세", "100세" 등 (보통 "N세만기"로 표현)
- 종신: "종신" (종신보험의 경우)

### 2. 납입기간 형식
- 연수 기반: "5년", "10년", "15년", "20년", "30년" 등 (보통 "N년납"으로 표현)
- 나이 기반: "60세", "65세" 등 (보통 "N세납"으로 표현)
- 전기납: "전기납" (보험기간 = 납입기간)
- 일시납: "일시납" (한 번에 전액)

### 3. 유효 조합 추출
- 보험기간/납입기간 매트릭스 테이블이 있는 경우:
  - 열 헤더 = 보험기간, 행 = 납입기간 (또는 반대)
  - 셀에 가입나이 등 값이 있으면 → **유효한 조합**
  - 셀이 "-", "해당없음", 비어있으면 → **무효 조합 (제외)**
- 테이블 없이 텍스트로 명시된 경우 → 해당 조합 그대로 추출

### 4. 상품별 구분
- 세부종목(1종/2종, 거치형/적립형 등)에 따라 다른 조합이면 각각 구분
- 모든 상품 공통이면 동일 조합 배열 적용

### 5. 제2보기개시나이 (연금상품인 경우)
- 연금상품에서 보기개시나이 또는 제2보기개시나이가 보험기간/납입기간과 연결되어 있으면 min/max 추출
- 비연금 상품이면 빈 문자열로 두세요

```json
      },

        "보험기간": "90세",
        "납입기간": "10년",
        "제2보기개시나이_min": "",

      }

  }

```

## 예시

문서에 아래 테이블이 있을 때:
| 구분 | 구분 | 90세만기 | 100세만기 |
|------|------|---------|----------|
| 일반가입형 | 5년납 | 만15~80세 | 만15~80세 |
| 일반가입형 | 10년납 | 만15~79세 | 만15~80세 |

→ 일반가입형의 유효 조합:
- (보험기간: "90세", 납입기간: "5년")
- (보험기간: "90세", 납입기간: "10년")
- (보험기간: "100세", 납입기간: "5년")
- (보험기간: "100세", 납입기간: "10년")

주의:
- 보험기간과 납입기간의 **원문 표현**을 그대로 사용하세요 (코드 변환하지 마세요).
- 상품 목록의 **모든 상품**에 대해 결과를 생성하세요.
- 같은 세부종목의 상품은 같은 조합을 가집니다.
