# Step 2a: 납입주기 추출

아래 문서에서 각 상품별 **납입주기(보험료 납입 빈도)** 정보를 추출하세요.

## 문서
```
{{DOCUMENT}}
```

## 상품 목록 (Step 1 결과)
```json
{{PRODUCT_LIST}}
```

## 추출 규칙

### 1. 납입주기 종류
문서에서 아래 납입주기를 찾으세요:
- **일시납** (한 번에 납부)
- **월납** (매월)
- **3개월납** (분기납과 동일)
- **6개월납** (반기납과 동일)
- **연납** (년납과 동일)

동의어 매핑:
- "분기납" → "3개월납"
- "반기납" → "6개월납"
- "년납" → "연납"
- "수시납" → "월납"

### 2. 납입주기 찾는 위치
- "납입주기" 섹션 또는 해당 키워드가 포함된 테이블/텍스트
- "보험료 납입주기: 월납" 같은 한 줄짜리 명시도 있음

### 3. 상품별 구분
- 모든 상품에 동일한 납입주기가 적용되면 → 모든 상품에 같은 주기 배열
- 특정 세부종목(예: "1종은 일시납, 2종은 월납")에 따라 다르면 → 해당 상품만 다른 주기 적용
- "단, ○○의 경우 일시납" 같은 단서 조항도 반영

### 4. "N년납" 주의
- "20년납", "5년납" 등은 **납입기간**이지 납입주기가 아닙니다. 이를 납입주기로 혼동하지 마세요.

## 출력 형식

상품 목록의 각 상품에 대해 납입주기명만 추출하세요. 코드 변환은 하지 마세요.

```json
[
  {
    "상품명": "한화생명 ○○보험 무배당 일반가입형 해약환급금 미지급형",
    "납입주기": [
      {"납입주기명": "월납"},
      {"납입주기명": "연납"}
    ]
  }
]
```

## 예시

문서에 "보험료 납입주기: 월납"이라고만 적혀 있는 경우:

```json
[
  {
    "상품명": "한화생명 H간병보험 무배당 일반가입형 해약환급금 미지급형",
    "납입주기": [{"납입주기명": "월납"}]
  },
  {
    "상품명": "한화생명 H간병보험 무배당 간편가입형(2년) 해약환급금 미지급형",
    "납입주기": [{"납입주기명": "월납"}]
  }
]
```

주의: 상품 목록에 있는 **모든 상품**에 대해 빠짐없이 결과를 생성하세요.
